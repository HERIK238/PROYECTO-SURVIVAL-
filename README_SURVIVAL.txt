PROYECTO-SURVIVAL- - Guía de Instalación
--------------------------------------------

1) Instala este proyecto en tu servidor local:
   - XAMPP:   C:\xampp\htdocs\PROYECTO-SURVIVAL-
   - Laragon: C:\laragon\www\PROYECTO-SURVIVAL-
   - Linux:   /var/www/html/PROYECTO-SURVIVAL-

2) Accede en el navegador:
   http://localhost/PROYECTO-SURVIVAL-/

3) Flujo de navegación:
   - index.php → redirige al login (views/auth.php).
   - views/auth.php → usa tu diseño original con css/style.css y js/script.js.
   - Al iniciar sesión pasa al dashboard (views/dashboard.php).

4) Archivos clave de login:
   - views/auth.php
   - css/style.css
   - js/script.js

5) Archivos del dashboard y estructura adicional vienen de Roepard,
   ya conectados y adaptados para PROYECTO-SURVIVAL-.

¡Listo para probar en tu servidor local!
